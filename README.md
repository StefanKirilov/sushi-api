# Sushi restaurant

Delivery sushi restaurant.

## Installation

### Prerequisites

Make sure you have the following software installed on your local machine:

- [Node.js](https://nodejs.org/) (which includes npm)
- [MongoDB](https://www.mongodb.com/)


### MongoDB
ÐœongoDB data must be taken from a folder "mongoDB_menu_collection".

### Install Server Dependencies
Navigate to the server directory and install the necessary dependencies:

cd server
npm install

### Install Client Dependencies
Navigate to the client directory and install the necessary dependencies:

cd client
npm install

### Running the Server
Navigate to the server directory and start the Express server:

cd server
npm start
The server will run on http://localhost:3030.

### Running the Client
Open a new terminal, navigate to the client directory, and start the React application:

cd client
npm start
The client will run on http://localhost:5173.
